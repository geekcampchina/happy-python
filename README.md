# happy-python

Happy-Python是一个简单易用的Python库，让编程轻松愉快

## 安装开发依赖

`pip install poetry pytest`

## 单元测试

运行单元测试：

`poetry run pytest`

## 安装happy-python

### Pip安装

`pip install happy-python`

## 本地安装

```bash
rm -rf dist/
poetry build -f wheel
pip install -U --user dist/happy_python-*-py3-none-any.whl
```
